***Welcome to PPLK+C (Predictor of Peptide Ligands of Potassium Channels)

Steps to run PPLK+C:

1- Extract all the PPLKC files

2- Install WinPython: WinPython64-3.7.1.0 available at: https://winpython.github.io/

3- Copy and paste all the PPLKC files into the installed WinPython software

4- Open IDLE (Python GUI) and go to: /File/Open/PPLKC_setup.py

5- Press F5
